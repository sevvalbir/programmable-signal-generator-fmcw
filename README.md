# Programmable Signal Generator with FMCW Generation

Bachelor's Graduation Project

---

## Overview

This project presents the design and implementation of a programmable signal generator capable of generating Frequency Modulated Continuous Wave (FMCW) signals.

The system was developed as my undergraduate graduation project in Electrical and Electronics Engineering.

The repository showcases the project architecture, hardware implementation, experimental results, and system overview.

The implementation source code is intentionally not shared publicly.

---

## Project Objectives

- Generate programmable waveforms
- Produce FMCW signals
- Adjustable frequency sweep parameters
- Embedded software implementation
- Hardware validation
